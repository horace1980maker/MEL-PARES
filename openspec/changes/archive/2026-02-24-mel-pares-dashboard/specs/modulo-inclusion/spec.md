## Requisitos AÑADIDOS

### Requisito: Matrices de inclusión por piloto

El módulo DEBE mostrar una matriz por cada piloto con las medidas de inclusión planificadas, su estado de implementación (checklist) y las evidencias asociadas.

#### Escenario: Ver matriz de inclusión de un piloto

- **WHEN** el usuario selecciona un piloto en el módulo de inclusión
- **THEN** se muestra una matriz con filas por medida de inclusión, columnas de estado (planificado/en progreso/completado) y enlaces a evidencias

### Requisito: Participación activa

El módulo DEBE registrar y mostrar indicadores de participación activa por piloto, incluyendo número de participantes, demografía y tipo de participación.

#### Escenario: Ver participación de un piloto

- **WHEN** el usuario consulta la participación activa de un piloto
- **THEN** se muestra un resumen con número de participantes, desglose demográfico y tipo de participación
